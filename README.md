# Aldagi Insurance Company

This is a collaborative front-end project replicating the website of **Aldagi Insurance Company**, created as part of the **project for the second stage of the "Women in AI" course** at **Business and Technology University**.

The objective was to build a responsive and visually appealing website using core front-end technologies, while collaborating effectively as a team using **ClickUp** for project management.

- **CSS3**

- **Swiper.js** – Used for the **slider**

## 📱 Responsive Design

The layout is fully responsive and adapts to the following breakpoints:

- **1440px** – Desktop (large screens)
- **1024px** – Laptop / small desktops
- **768px** – Tablets
- **480px** – Mobile devices

Each breakpoint ensures that the layout remains user-friendly and visually appealing across different screen sizes.
